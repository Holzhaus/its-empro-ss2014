# Übung 11
## Frage 3 (5 Punkte)

In welchem Sleep Mode kann der Controller durch ein "Other I/O" Event aufgeweckt werden? 
		
[X] Idle
[ ] Power Save
[ ] Standby
[ ] Power Down
[ ] ADC Noise Reduction

### Lösung

Sleep Mode: Idle
