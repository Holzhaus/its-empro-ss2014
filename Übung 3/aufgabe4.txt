Bit 1: Interrupt Flag

Bit 2: Bits können aus dem Status Register in ein anderes Register geschrieben werden (aus einem Register in das Stauts Register funktioniert auch).

Bit 3: Half Carry Flag

Bit 4: Flag für Vorzeichentest

Bit 5: Zweierkomplement Overflow indicator

Bit 6: Negativ-Vorzeichen Flag

Bit 7: Zero Flag für CPI Operationen

Bit 8: Carry Flag wird gesetzt wenn eine Rechnung zu groß für die Register ist
